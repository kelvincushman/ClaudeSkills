# Atopile Saving Your Work

This skill explains the recommended workflow for saving and versioning Atopile projects using Git.

## Core Workflow

1.  **Initialize Git**: `git init`
2.  **Add Files**: `git add .`
3.  **Commit**: `git commit -m "Your message"`
